# for-get-Routine
不背套路——武术拳操练习工具，通过刷题的方式记忆套路动作。使用方法：应用顶部显示当前所在段位，下方卡片展示当前动作。答题时从四个选项中选出正确的下一个动作；若当前动作需要发声，则先选择"哈"或"嘿"。支持拖动进度条从任意动作开始练习，适合零碎时间反复训练，直到套路烂熟于心。
